The Million Dollar Projects

Objective:

Design and implementation of a unified Network Command Console (“NOCTIS”) to centralize monitoring, cybersecurity, access management, and operational analytics across large-scale enterprise IT ecosystems.

Scope & Architecture

Core Framework: MERN stack (MongoDB, Express, React, Node.js) with GoLang microservices for low-latency network packet handling.

Modules

Network Monitoring Dashboard:
Real-time telemetry ingest and visualisation of traffic flows across internet, intranet, and VPN endpoints.

Threat Detection & Incident Response:
AI-driven anomaly detection pipeline integrating predictive ML classifiers for intrusion prevention, with incident orchestration triggered via role-based alerts.

Access Control & Authentication:
Zero Trust architecture including Single Sign-On (SSO), Multi-Factor Authentication (MFA), and fine-grained RBAC provisioning.

Packet Tracing Engine:
Live graph-based tracing of TCP/UDP packets, with heuristics for anomaly flagging and latency bottleneck detection.

IoT & RFID Integration:
Device onboarding protocols, RFID-based entry management, and automated security approval workflows.

Audit & Compliance Logging:
Immutable log chains to satisfy SOC 2, ISO 27001, and GDPR audit trails.

Technical
Innovations

Design Considerations:

Temporal-spatial collision detection for double bookings, leveraging live calendar sync & permission trees.

Story Behind the Build:

Behavioral interaction layer that dynamically modifies user funnels in response to data trigger thresholds.

Challenges:

Geo-data inference engine for auto-suggested localization, eliminating manual language pre-selection.
